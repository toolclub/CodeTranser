input_schema JSON Schema,前端表单渲染 + AI 入参校验

字段名类型必填默认说明
object — 待校验的报文主体
string — 预期 hash,十六进制
enum "sha256" 摘要算法

output_schema

字段名类型说明
matchbool是否匹配
computed_hashstring实际计算出的 hash,便于排查

出边语义 edge_semantics 告诉代码生成器每条出边对应实例的哪个字段

✓
on_match
对应 C++ 实例字段:Node* next_on_match
0..1
✕
on_mismatch
对应 C++ 实例字段:Node* next_on_mismatch
0..1

变量引用:{{fields}} {{input}} {{ctx.table_data}}。Prompt 会自动附加 output_schema 做 JSON 结构约束。
