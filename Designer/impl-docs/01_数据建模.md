# 01 · 数据建模

> 依赖:无
> 交付物:
> - 全部表的**纯 DDL**(`config/sql/01.ddl`)
> - SQLAlchemy 2.0 ORM(仅用于 CRUD,**不用来建表**)
> - MongoDB 集合结构定义 + TypedDict
> - 领域值对象(节点模板 / 节点实例 / Bundle / 边 / 森林 / 运行状态)
> - DTO(pydantic v2)
> - 迁移 runner 规范
>
> 验收:
> 1. `mysql < config/sql/01.ddl` 在空库上执行完毕,16+2 张表建好;再跑 `02.ddl` ... 可挨着迭代
> 2. `python -m app.cli migrate` 自动按字典序跑 `config/sql/*.ddl`,已跑过的跳过
> 3. `pytest tests/unit/domain` 全绿
> 4. 预置的元模板(`t_meta_node_template` 初始数据)可被 admin 读出

## 1.1 核心概念层级

> 这是本次整体建模的基石。**所有章节与这里的术语对齐**。

```
节点模板 (t_node_template 一行 = Tool;前端称 node-template)
 ├─ 设计人员按"元模板"填出来的节点抽象
 ├─ 一种节点类型一个模板
 ├─ 元模板(t_meta_node_template)告诉设计人员"节点模板要填哪些字段"

节点实例 (NodeInstance;小节点,代码层 = 代码片段)
 ├─ 引用某个节点模板
 ├─ 带自己的 field_values
 ├─ 一个节点模板在同一张图里可以被实例化任意多次
 ├─ 可归属某个 Bundle,也可以游离(孤儿)

Bundle (大节点;代码层 = 一个 class/function)
 ├─ 若干节点实例的容器
 ├─ 纯视觉/组织分组,本身不参与执行
 ├─ 代码生成时:每个 Bundle → 一个 class/function,内部小节点 = 片段被内联
 ├─ v1 不嵌套

边 (Edge)
 ├─ 连接两个节点实例(可跨 Bundle)
 ├─ Bundle 本身不连线
 ├─ 边的 semantic 必须是源节点模板的 edge_semantics[*].field 之一

DAG (视图,不存储)
 ├─ 从任意 root(入度 0 的节点实例)出发沿边走到底 = 一个 DAG
 ├─ 可横跨多个 Bundle
 ├─ 一个森林里通常有若干个 DAG

森林 (CascadeForest / 一张图的全部)
 ├─ 若干 Bundle + 若干游离节点实例 + 全局 edges
 ├─ 就是前端画布保存的那个"大 JSON"
 ├─ 不含 position(渲染交前端 antv)
```

## 1.2 约定与命名

### 1.2.1 表名

- 全部 `t_` 前缀,`snake_case`,单数(`t_user` / `t_workflow_run`)
- 字符集 `utf8mb4` + `utf8mb4_0900_ai_ci`
- 引擎 `InnoDB`
- 主键:**业务/运行实体**用 `VARCHAR(32)`(带前缀的 string id);**基础字典类**用 `BIGINT AUTO_INCREMENT`

## 1.3 config/sql 目录

```
config/sql/
├── 01.ddl          初始建表 + 预置元模板 + 迁移追踪表
├── 02.ddl          后续迭代(每次新功能 / 字段变更一个新文件)
├── 03.ddl
└── ...
```

**规矩**:
- 文件名严格 `NN.ddl`(两位数字 + 点 + `ddl`)
- 文件名**只递增、不改历史**(改已发布的 ddl = 灾难)
- 每个 ddl 第一行注释写 `-- NN.ddl: purpose`
- 都用**幂等**语句:`CREATE TABLE IF NOT EXISTS`、`ALTER TABLE ... ADD COLUMN IF NOT EXISTS`(MySQL 8+)
- 一个 ddl 内多个语句用 `;` 分隔

### 1.3.1 迁移追踪表(必建,第一张)

```sql
CREATE TABLE IF NOT EXISTS t_migration_applied (
  file_name   VARCHAR(128) PRIMARY KEY,
  checksum    VARCHAR(64)  NOT NULL,
  applied_at  DATETIME(6)  NOT NULL DEFAULT CURRENT_TIMESTAMP(6)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COLLATE=utf8mb4_0900_ai_ci;
```

runner 逻辑:
1. 扫描 `config/sql/*.ddl` 按文件名排序
2. 对每个文件算 sha256
3. 若 `t_migration_applied` 中存在同名记录:
   - checksum 一致 → 跳过
   - 不一致 → 报错终止(历史 DDL 被改,人工介入)
4. 不存在 → 执行所有 statements,成功后插入 `t_migration_applied`
5. 第一个 `01.ddl` 自身建 `t_migration_applied`(语句 `IF NOT EXISTS` 幂等)

### 1.3.3 迁移 runner(Python)

```python
# app/infra/migrate/runner.py
import hashlib
import asyncio
from pathlib import Path
from sqlalchemy import text
from app.config import Settings
from app.infra.db.session import create_engine
from app.infra.logging import get_logger

log = get_logger(__name__)
SQL_DIR = Path(__file__).resolve().parents[3] / "config" / "sql"

class MigrationError(Exception): ...

async def run_migrations(settings: Settings, sql_dir: Path | None = None) -> list[str]:
    """按字典序跑 config/sql/*.ddl。
    返回本次实际执行的文件名列表(跳过的不返回)。"""
    base = sql_dir or SQL_DIR
    files = sorted(base.glob("*.ddl"))
    if not files:
        raise MigrationError(f"no .ddl under {base}")

    engine = create_engine(settings)
    executed: list[str] = []

    async with engine.begin() as conn:
        # 先确保追踪表存在(第一个 ddl 会建;但为了 runner 启动顺序,这里也容忍 race)
        await conn.execute(text("""
            CREATE TABLE IF NOT EXISTS t_migration_applied (
              file_name   VARCHAR(128) PRIMARY KEY,
              checksum    VARCHAR(64)  NOT NULL,
              applied_at  DATETIME(6)  NOT NULL DEFAULT CURRENT_TIMESTAMP(6)
            ) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COLLATE=utf8mb4_0900_ai_ci
        """))

    for f in files:
        body = f.read_text(encoding="utf-8")
        checksum = hashlib.sha256(body.encode("utf-8")).hexdigest()
        async with engine.begin() as conn:
            row = (await conn.execute(text(
                "SELECT checksum FROM t_migration_applied WHERE file_name = :fn"
            ), {"fn": f.name})).first()
            if row:
                if row[0] != checksum:
                    raise MigrationError(f"ddl {f.name} checksum changed; history is immutable")
                log.info("migration_skip", file=f.name)
                continue
            log.info("migration_apply", file=f.name)
            # MySQL 不支持 transactional DDL;逐条执行
            for stmt in _split(body):
                if stmt.strip():
                    await conn.execute(text(stmt))
            await conn.execute(text(
                "INSERT INTO t_migration_applied (file_name, checksum) VALUES (:fn, :cs)"
            ), {"fn": f.name, "cs": checksum})
            executed.append(f.name)

    await engine.dispose()
    return executed

def _split(sql: str) -> list[str]:
    """简易 SQL 分号拆分。容忍:
      - 行注释 -- ... \n
      - 不处理字符串里带 ; (v1 的 DDL 本身不涉及)
    """
    out: list[str] = []
    buf: list[str] = []
    for line in sql.splitlines():
        stripped = line.strip()
        if stripped.startswith("--"):
            continue
        buf.append(line)
        if stripped.endswith(";"):
            out.append("\n".join(buf)[:-1])  # 去掉末尾分号
            buf = []
    tail = "\n".join(buf).strip()
    if tail:
        out.append(tail)
    return out
```

**关键约束**(校验器强制,见 04 章):

1. `bundles[].node_instance_ids` 必须是 `node_instances[].instance_id` 的子集
2. 一个 `instance_id` **最多归属一个 bundle**(互斥);没出现在任何 bundle 的就是"游离节点"
3. `edges[].from/to` 必须是 `node_instances[].instance_id` 之一
4. `edges[].edge_semantic` 必须在源节点的 `template_snapshot.edge_semantics[*].field` 里
5. 森林整体必须**无环**(拓扑可排)
6. **不存 DAG**:DAG 是 04 章 `DagComputeVisitor` 从 edges 计算出来的视图

**粘贴/复制语义**(前端 ctrl+c/v):

- 前端把一个 bundle(或一组节点实例 + 相关边)序列化成 JSON → 剪贴板
- 粘贴时前端**重新生成** `bnd_*` / `n_*` / `e_*` 全部 id,避免冲突
- 跨图粘贴同理,前端处理;后端只认最终"整张森林 JSON"

```python
from abc import ABC, abstractmethod
from dataclasses import dataclass
from typing import Any, Mapping
from app.domain.tool.tool import NodeTemplate

class GraphNode(ABC):
    @abstractmethod
    def accept(self, v: "ForestVisitor") -> Any: ...

@dataclass(frozen=True, slots=True)
class NodeInstance(GraphNode):
    instance_id:       str
    template_snapshot: NodeTemplate    # 整份模板快照
    instance_name:     str
    field_values:      Mapping[str, Any]
    bundle_id:         str | None = None   # 归属的 bundle;None = 游离
    def accept(self, v): return v.visit_node(self)

@dataclass(frozen=True, slots=True)
class Edge:
    edge_id:  str
    src:      str             # instance_id
    dst:      str             # instance_id
    semantic: str
    def accept(self, v): return v.visit_edge(self)

@dataclass(frozen=True, slots=True)
class Bundle(GraphNode):
    bundle_id:         str
    name:              str
    description:       str
    node_instance_ids: tuple[str, ...]
    def accept(self, v): return v.visit_bundle(self)

@dataclass(frozen=True, slots=True)
class CascadeForest(GraphNode):
    graph_version_id:    str
    version_number:      int
    bundles:             tuple[Bundle, ...]
    node_instances:      tuple[NodeInstance, ...]   # 全部节点实例(不管在不在 bundle)
    edges:               tuple[Edge, ...]
    metadata:            Mapping[str, Any]

    # 便捷查询
    def node_by_id(self, iid: str) -> NodeInstance:
        for n in self.node_instances:
            if n.instance_id == iid: return n
        raise KeyError(iid)
    def bundle_by_id(self, bid: str) -> Bundle:
        for b in self.bundles:
            if b.bundle_id == bid: return b
        raise KeyError(bid)
    def orphans(self) -> list[NodeInstance]:
        return [n for n in self.node_instances if n.bundle_id is None]

    def accept(self, v): return v.visit_forest(self)
```

## 1.9 SQLAlchemy ORM(用于 CRUD,不建表)

```python
# app/domain/errors.py
class DomainError(Exception):
    code: str = "INTERNAL_UNEXPECTED"
    http_status: int = 500
    def __init__(self, message: str = "", **extra):
        super().__init__(message)
        self.message = message or self.code
        self.extra = extra

class Forbidden(DomainError):            code = "PERM_DENIED";            http_status = 403
class Unauthorized(DomainError):         code = "AUTH_UNAUTHORIZED";      http_status = 401
class Conflict(DomainError):             code = "CONFLICT_STALE_VERSION"; http_status = 409
class TooManyRequests(DomainError):      code = "RATE_LIMITED";           http_status = 429

class BusinessError(DomainError):        http_status = 422
```
