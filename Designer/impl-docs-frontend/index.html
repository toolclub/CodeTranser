级跳设计平台 · 前端原型

Vue 3 + Pinia + Vite + antv X6。后端已有 00–07 实施文档,本页是前端 UI 的高保真视觉契约。

11 个页面 antv X6 画布 无前端鉴权(后端装饰器判) 大 JSON 进 / 大 JSON 出

图列表 核心

CascadeGraph 列表,展示最近改动 / 版本 / 最近一次 Run 状态,支持新建、搜索、归档。

画布编辑器 核心

antv X6 真实交互。左侧节点库,中间自动布局的 bundle + 游离节点 + 跨 bundle 边,右侧属性面板。

节点模板编辑器

元模板驱动表单 · description / input_schema / output_schema / edge_semantics,右侧实时 JSON 预览。

节点模板库

按分类浏览全部节点模板,显示 engine、owner、版本数;设计人员入口(私有)+ admin 入口(全局)。

场景编辑器

两列 JSON 编辑器:input / expected_output。多场景,一键触发 Phase1-only 或完整流水线。

Run 列表

按图 / 状态筛选,展示 pipeline variant、耗时、阶段进度条。实时 WS 更新。

Run 详情 · 三阶段时间线 核心

Phase1 (结构 + 场景) / Phase2 (代码) / Phase3 (沙箱) 时间线 + 每步 trace + LLM 归因。

代码 diff

Phase2 产物与上一次快照的 unified diff。左侧文件树,右侧 diff2html 渲染。

Review

复核人员对图 / 代码逐节点评论,线程式对话,批准/打回按钮。

无权限 / 错误态

403 / 500 / 网络错误的统一兜底页。后端装饰器决定真正鉴权,前端只负责展示。
使用方式:直接在浏览器里打开本页(impl-docs-frontend/index.html),点击卡片进入每个页面。 画布编辑器页面使用 antv X6 CDN,可以真实拖拽连线;其他页面是静态高保真视觉。 需要离线可在 assets 目录下缓存 CDN 资源,但默认走 unpkg。

页面 / 后端 API 对应: 图列表 ↔ GET /api/graphs;画布 ↔ GET/POST/PUT /api/graphs/:id/versions; 节点模板编辑器 ↔ POST /api/node-templates 或 /api/admin/node-templates; 场景 ↔ POST /api/runs { scenarios, variant };Run 详情 ↔ GET /api/runs/:id + WS。 所有鉴权返回 401/403 统一跳 无权限页。
